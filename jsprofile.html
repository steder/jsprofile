This page demonstrates the simplest bare-bones usage of jsprofile with the bundled default profiler view.

You might also want to try the functional tests. The external javascript files will need to be hosted by a web server, as they are fetched with AJAX. If using Apache, configure DocumentRoot to be the directory containing this HTML file, and navigate to it via http://localhost/jsprofile.html. Alternatively, copy the jsprofile directory to somewhere under your htdocs.
